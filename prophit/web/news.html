Stay Informed
There are two mailing lists you can use to stay informed about prophIt:
  • prophit discussion. We're interested in your feedback and comments. We'll also use this list to make announcements about prophIt releases and other developments. To post to the list, click here  To join the list, click here.
  • prophit dev. This list is for the developers working on prophIt. This is an appropriate place to send bug reports, for example, or to ask questions. We reserve the right to cross-post back to the main discussion list if we think the answer is of general interest. To send a message to the prophit dev list, click here  This list is for developers of prophIt. The list info page is here.
If you are interested in contributing to the development of prophIt, send email to prophit-dev@prophit.westslopesoftware.com

Latest News

[2/14/2002] prophIt Beta-2. New features implemented: Intelligent Labeling, Wireframe mode, Level of Detail. In addition, rendering of selected blocks has been improved, and the depth slider has been enhanced to automatically adjust for the stack depth.
