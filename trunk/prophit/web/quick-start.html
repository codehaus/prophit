Get Orbit

Install Orbit

  • Read about Running Orbit and get JavaWebStart
  • Use JavaWebStart to install Orbit

Open the Hello List -prof Sample Profile

  • Read Orbit Concepts for a quick overview of how the Orbit UI works.
  • Use the menu item File/Sample Profiles/prof/HelloList to open the Hello List java -prof sample profile.
  • To view your own profile, run your Java program using one of the supported profilers. Then load the profile into Orbit using the File/Open menu item.
