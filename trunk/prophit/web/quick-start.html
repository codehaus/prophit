Quick Start

This is the quick way to get started.

Get prophIt

Install prophIt

  • Go to the download page to get JavaWebstart and then install prophIt.

Open the Hello List -prof Sample Profile

  • Read prophIt Concepts for a quick overview of how the prophIt UI works.
  • Use the menu item File/Sample Profiles/hprof/HelloList to open the Hello List java hprof sample profile.

To View Your Own Profile

  • Run your Java program using one of the supported profilers. Then load the profile into prophIt using the File/Open menu item.
