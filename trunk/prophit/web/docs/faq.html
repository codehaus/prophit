Frequently Asked Questions (FAQ)

How do I get prophIt?

Go to the download page to get the details on the pre-requisites for running prophIt, and to read the prophIt Evaluation license. After agreeing to the licensing terms, you may download the product, using Sun's Java WebStart technology. This let's you easily download all the application files you need in a format that is digitally signed by Verisign as proof that the content was produced by Westslope Software. The installation is trivial. You can also install a desktop shortcut to launch the application in the future. Each time you launch prophIt, the latest updates can be installed to keep you on top of the latest and greatest.

How do I know I'm not downloading a virus?
We have invested in a digital certificate from Verisign to validate that the package you download is the package that we put on our servers for distribution (if the contents are changed, the digital signature will be altered). We believe it is important to be responsible distributors of software over the Internet, and signed up with Verisign so that you, our customers and users, could feel confident that you are downloading a legitimate application associated with a legitimate legal entity. If you have any questions about how this works, please consult Verisign's website for all the details on digital signatures. Or, feel free to contact us with questions.
Why do I have to use Java WebStart?
Java WebStart accomplishes two things
  1. Installation is as easy as hitting a web page. Really. The WebStart app downloads the prophIt application and runs it using your installed Java VM. We simply include all the dependencies for our application in our packaging.
  2. Application updates can be retrieved anytime you run a Java WebStart app. This makes it easy to distribute bugfixes and enhancements to users of our application.
We think those two benefits make Java WebStart a real bonus for us, and for our users.
How much does prophIt cost?
The evaluation license for prophIt is currently FREE. However, this license comes with some limitations. We intend to continue to make prophIt free for individual non-commercial use. If you have a commercial application for prophIt, please contact us to discuss licensing.
Can I use prophIt to view profiling data for programs written in C, C++ or some other language other than Java?
prophIt is built to be flexible about reading in data through adapters. Currently, adapters exist for data generated by the profiles on the supported profilers page. At present, there are no adapters written to interpret data written by performance profilers for other languages and platforms. However, we anticipate supporting the Python and Perl profiler data formats in the future.
What 3rd Party libraries does prophIt use?
We currently use GL4Java, log4jME, nanoxml, and lp_solve. At a future date we'll update this entry with links to these libraries.
Why is the download button a link to prophIt.jnlp? My browser doesn't know what that is, how do I get a zip file?
We use Java Web Start to deploy our software. We think it provides the best way to get updated installations to our user base, and the easiest way to simplify installation. If you have the most recent versions of Java, Web Start comes included. If not, you can visit Sun's Java website to pick up Web Start.

If you once had Java Web Start installed, but since then deleted the installation, Windows may believe Web Start is still installed and have trouble running the prophIt application. We assure you that if Java Web Start is installed correctly, prophIt will run very smoothly. It will even pick up updates and patches as needed. If your Web Start installation appears to be broken, re-install it, and then try to download prophIt again, and we think you'll meet with success.

Finally, a note on zip files. We distribute our application as a Java Web Start app because this allows digital signature of the application and its pieces, ensuring that you have a valid application. At this time, we do not intend to distribute a zip file of the source or the binaries.
