Glossary

Previous : Sample Profiles   Up : Documentation   Next :
caller
A function which has invoked the current function. The caller function is displayed as the block underneath the current block in the tower diagram. The inclusive time of a function is always less than or equal to the inclusive time of its caller.
callee
A function which has been invoked by the current function. Callee functions are displayed as the blocks stacked on top of the current block in the tower diagram. The sum of the inclusive times of the callees of a function is always less than or equal to the inclusive time of that function.
depth slider
The slider control at the top of the Orbit window. The depth slider controls how many levels of the call graph below the diagram root are shown in the tower diagram.
diagram root
The function which is rendered as the base of the tower diagram
exclusive time
Amount of time spent in a function, not counting the time spent in its callees. The exclusive time is the sum of the time spent in all the statements and instructions of the function itself. A function with a large exclusive time is commonly referred to as a 'hotspot'.
 
inclusive time
Amount of time spent in a function, including the time spent in its callees.
tower diagram
Refers to the visualization technique invented for and used by Orbit. See this screenshot.
selected call
The block (function) in the tower diagram which has been selected by the user. The selected call may be set by single-clicking on a block in the tower diagram with the mouse, or by double-clicking on a row in a call table.
