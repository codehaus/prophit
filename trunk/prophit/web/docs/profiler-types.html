Supported Profilers

prophIt is a visualization tool for profile data; it does not create the profile data itself. prophIt supports loading profiler data from 2 different profilers: java -prof and hprof.

To use prophIt, you first run your Java program in one of these profilers to generate a data file. Then you can use the File/Open command in prophIt to load and display that data file. prophIt is for CPU profiling only, it does not do memory or monitor profiling.
