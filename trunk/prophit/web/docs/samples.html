prophIt Samples
We've provided discussion of a few samples here, and links to these samples from within the prophIt application as well.
The HelloList sample program
HelloList.java is a simple Java program which compares the performance of:
  • String and StringBuffer
  • ArrayList and LinkedList

The program runs four loops, each of which tests one combination of String and Collection classes:

Sample Description
arrayAndString For each iteration, builds an ArrayList of Strings, each of which is created by using the += operator to concatenate the characters in the phrase Hello World
arrayAndBuffer For each iteration, builds an ArrayList of Strings, each of which is created by using a StringBuffer to concatenate the characters in the phrase Hello World
linkedAndString For each iteration, builds an LinkedList of Strings, each of which is created by using the += operator to concatenate the characters in the phrase Hello World
linkedAndBuffer For each iteration, builds an LinkedList of Strings, each of which is created by using a StringBuffer to concatenate the characters in the phrase Hello World

HSQLDB Example

HSQLDB is the hqsl Database Engine, an Open Source Java database. The sample profile in prophIt may be created by running the script hsqldb_v1.61/demo/run.bat with Java version 1.4.0-b92 on Windows 2000 with the following Java command-line options:

 java -Xrunhprof:cpu=samples,depth=6
