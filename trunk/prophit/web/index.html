About Orbit

Orbit is an innovative visualization tool for working with Java profiles. Orbit does not generate profile data itself. Rather it reads data files generated by a supported profiler and presents them in a powerful user interface.

Running Orbit

Runtime Environment

Orbit is implemented using the Java Swing GUI library, using the GL4Java Java binding to OpenGL to render graphics-intensive parts of the UI. Windows is the only currently supported platform, but OpenGL is available for other platforms as well and Orbit will be supported on them as well.

JavaWebStart

Orbit uses JavaWebStart (version 1.0.1_02 or later) for installation and automatic upgrades. To run Orbit, first you need to obtain JavaWebStart:
  • If you already have the JDK 1.4, you already have JavaWebStart! It is bundled with the J2SE
  • Otherwise, you will need to download and install JavaWebStart separately. Go to the JavaWebStart and follow the installation instructions there.

Java Version

Orbit requires JDK 1.3 or higher. It will run and look better with JDK 1.4+.
