Supported Profilers

Previous : Orbit Documentation   Up : Orbit Documentation   Next : Orbit Samples

Orbit is a visualization tool for profile data; it does not create the profile data itself. Orbit supports loading profiler data from 2 different profilers: java -prof and hprof.

To use Orbit, you first run your Java program in one of these profilers to generate a data file. Then you can use the File/Open command in Orbit to load and display that data file. Orbit is for CPU profiling only, it does not do memory or monitor profiling.
