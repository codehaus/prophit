Runtime Environment
prophIt is implemented using the Java Swing GUI library, using the GL4Java Java binding to OpenGL to render graphics-intensive parts of the UI. Windows is the only currently supported platform, but OpenGL is available for other platforms as well. We expect to offer supported versions of prophIt on these platforms also.
Pre-Requisites

Java Webstart

prophIt uses JavaWebStart (version 1.0.1_02 or later) for installation and automatic upgrades. To run prophIt, first you need to obtain JavaWebStart:

  • If you already have the JDK 1.4, you already have JavaWebStart! It is bundled with the J2SE
  • Otherwise, you will need to download and install JavaWebStart separately. Go to the JavaWebStart home page and follow the installation instructions there.

Java Development Kit

prophIt requires JDK 1.3 or higher. It will run and look better with JDK 1.4.0 or higher. You can download the JDK from Sun.
